# Hermes Self-Improvement Loop

Free improvements to keep enabled:

1. Keep secrets out of config; use `.env` references.
2. Enable `security.redact_secrets` and `privacy.redact_pii`.
3. Keep `AGENTS.md` current for project context.
4. Use `SOUL.md` for stable Telegram-first style.
5. Prefer built-in `web_search`/`web_extract` over paid search MCP.
6. Run no-agent cron jobs for monitoring/backups/scans to avoid token spend.
7. Save reusable workflows as skills; patch stale skills immediately.
8. Keep optional skills lightweight: FastMCP and mcporter are useful; avoid paid/heavy integrations without a clear need.
